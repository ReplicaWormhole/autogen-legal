Autogen Privacy Policy

Last updated: June 7, 2026

1. Overview

Autogen is a local desktop tool for creating educational videos and, when authorized by the user, uploading those videos to TikTok through TikTok Login Kit and the TikTok Content Posting API.

2. Information Processed

Autogen may process video project data such as paper metadata, generated scripts, captions, prompts, audio files, images, rendered videos, verification reports, and posting receipts. When TikTok publishing is enabled, Autogen may also process TikTok OAuth data, including an access token, refresh token, open ID, granted scopes, and token expiration timestamps.

3. Local Storage

TikTok OAuth tokens and generated video artifacts are stored locally on the user's machine, unless the user separately configures external services or publishes content. The app is designed to use TikTok tokens only to refresh authorization, query creator posting options, upload videos, and check publishing status for the authorized account.

4. Sharing

Autogen does not sell personal information. It does not share TikTok OAuth tokens with third parties except as necessary to call TikTok APIs requested by the user. Generated media and captions are sent to TikTok only when the user initiates or configures publishing.

5. Third-Party APIs

Depending on user configuration, Autogen may call services such as TikTok, ElevenLabs, Replicate, arXiv, or other APIs. Data sent to those services is handled under their respective privacy policies and terms.

6. Data Retention and Deletion

Because Autogen stores its working files locally, the user controls retention. The user can delete generated outputs, logs, and stored OAuth token files from the local project directory. The user can also revoke TikTok authorization from TikTok account settings.

7. Security

Autogen stores TikTok token bundles in a local file with restricted file permissions where supported by the operating system. Users should keep their device, repository, and environment files private and should not commit secrets or token files to public repositories.

8. Contact

For questions about this privacy policy, contact the repository owner through the GitHub account hosting this page.
