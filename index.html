Educational video generation and publishing tool

Autogen

Autogen is a local desktop tool for creating educational short-form videos about arXiv physics papers. It helps generate scripts, AI-assisted visuals, voice narration, rendered videos, captions, and verification reports.

When a user authorizes a TikTok account, Autogen can use TikTok Login Kit and the TikTok Content Posting API to upload videos to that authorized account. Publishing remains controlled by the user's local configuration and TikTok account authorization.
